# Prospera Distributeur

Plateforme AI-native de pilotage commercial pour distributeurs et grossistes — Togo (SNB, Zener, Tampico…).

## Démarrage

```bash
cd prospera-distributeur
npm install
npm run dev
```

Application : **http://localhost:3002**
